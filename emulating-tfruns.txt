Process for emulating tfruns

1. Look for train.py and verify existence
2. Initialize a rundir
3. Perform run
4. Prepare run to return (return_runs(run_record(run_dir)))
5. return run (return_record(run_dir))
6. Save run (save_run_view())


Main Pieces:
- initialize_run()
- unique_run_dir()
- clear_run()
- write_run_metadata()
- reset_tf_graph()
- do_training_run()
- run_record()
- return_runs()
